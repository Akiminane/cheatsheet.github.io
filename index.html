HTML and CSS Tags/selectors

Tous les Tags HTML et CSS utilisés pour la création de ce site web

HTML

Tag Description
<head> Fournit des informations générales (métadonnées) sur le document, incluant son titre et des liens ou des définitions vers des scripts et feuilles de style.
<body> Représente le contenu principal du document HTML
<header> Représente du contenu introductif, généralement un groupe de contenu introductif ou de contenu aidant à la navigation.
<nav> Représente une section d'une page ayant des liens vers d'autres pages ou des fragments de cette page.
<ul> Représente une liste d'éléments sans ordre particulier.
<ol> Représente une liste ordonnée.
<li> Utilisé pour représenter un élément dans une liste.
<h1>...<h6> Représentent les six niveaux de titre de section. <h1> correspond au niveau de section le plus haut et <h6> correspond au niveau le plus faible.
<p> Représente un paragraphe de texte.
<table> Permet de représenter un tableau de données, c'est-à-dire des informations exprimées sur un tableau en deux dimensions.
<thead> Définit un ensemble de lignes qui définit l'en-tête des colonnes d'un tableau.
<tbody> permet de regrouper un ou plusieurs éléments <tr> afin de former le corps d'un tableau HTML
<tr> Définit une ligne de cellules dans un tableau.
<th> Définit une cellule d'un tableau comme une cellule d'en-tête pour un groupe de cellules.
<td> définit une cellule d'un tableau qui contient des données.
<div> est le conteneur générique du contenu du flux. Il n'a aucun effet sur le contenu ou la mise en page tant qu'il n'est pas mis en forme d'une manière quelconque à l'aide de CSS.
<footer> Représente le pied de page de la section ou de la racine de sectionnement la plus proche.

CSS

Fonction Description
margin Définit la taille des marges sur les quatre côtés de l'élément.
padding Permet de définir les différents écarts de remplissage sur les quatre côtés d'un élément
background-color Permet de définir la couleur utilisée pour l'arrière-plan d'un élément
color Définit la couleur de premier plan d'un élément texte et de ses éventuelles décorations.
@font-face Permet de définir une police d'écriture particulière à utiliser pour afficher le texte de pages web.
font-family Permet de définir une liste, ordonnée par priorité, de polices à utiliser pour mettre en forme le texte de l'élément ciblé.
font-size Définit la taille de fonte utilisée pour le texte.
font-style Permet de sélectionner une fonte italique (italic) ou oblique (oblique)
width Permet de définir la largeur de la boîte du contenu d'un élément.
height Définit la hauteur de la boîte de contenu d'un élément.
position Définit la façon dont un élément est positionné dans un document.
display définit le type d'affichage utilisée pour le rendu d'un élément (de bloc ou en ligne) et la disposition utilisée pour ses éléments fils : grille ou boîtes flexibles.
border permet de définir les propriétés liées à la bordure. border peut être utilisée pour définir les valeurs de border-width, border-style et border-color.
text-dedcoration Décore le texte en ajoutant une ligne pouvant être positionnée sous, sur ou à travers le texte.
